# Jornada Python
Projetos práticos em Python explorando automação, análise de dados, machine learning e chatbots.

Este repositório reúne projetos práticos desenvolvidos durante minha jornada de aprendizado em Python ao longo do mini curso "Jornada Python" da Hashtag Treinamentos, realizado em março de 2026.

A proposta é explorar diferentes aplicações da linguagem através de projetos reais.
